# LB Flavor Gradient — Sliders Build

This build matches your preview: LB branding + per‑flavor stop sliders + smudge option + export dropdown, with logo included.

## Run locally
```bash
npm i
npm run dev
```

## Build for the web
```bash
npm run build
npm run preview
```

Deploy `dist/` to Netlify, Vercel, or GitHub Pages.
